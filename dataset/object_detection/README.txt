Aerial Maritime Detection Dataset: YOLO Version (compatible with v5 and v8)
===========================================================================
This dataset contains over 2000 labelled aerial images of maritime locations for object detection.

 - This dataset is automatically generated as per your student ID and it is unique to you.
 - Dataset provided is already split into 3 parts.
 - Please check and update the data.yaml file provided as per the correct path.
 - This is not compatible for YOLO-X as it requires other annotation format (PASCAL VOC).
